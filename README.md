# simple-api-express

A simple api working with express middleware on nodejs server.

## Technologies
- [expressjs](http://expressjs.com/)
- [jsonwebtoken](https://github.com/auth0/node-jsonwebtoken)
- [nanojs](https://github.com/apache/couchdb-nano)

## Comands
- To compile
```
npm run compile
```
- To production
```shell
npm run app
```
- To development
```
npm run dev
```
- Only test
```
npm run test
```
